# Robotik-Schach Stack (Moveo + FluidNC)

This folder contains a two-host stack for a Moveo 6DOF chess robot:

- `edge/`: hardware-near control, safety, vision, and deterministic skills
- `agent/`: learning, chess engine integration, REST API
- `common/`: shared schemas and protocol docs
- `specs/`: MVP and acceptance tests
